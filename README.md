# 🌌 Raka Fahlevi Portfolio Website

Website portofolio pribadi yang dirancang untuk menampilkan profil, pengalaman, keterampilan, publikasi ilmiah, pencapaian akademik, serta proyek-proyek yang pernah dikerjakan oleh Raka Fahlevi.

Website ini mengusung tema **Astronomy & Space Exploration** dengan konsep visual modern menggunakan efek glassmorphism, animasi interaktif, dan nuansa kosmik yang merepresentasikan perjalanan eksplorasi dalam dunia teknologi dan data.
